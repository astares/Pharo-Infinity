## Quick Start
### Installation

```Smalltalk
Metacello new 
    repository: 'github://astares/Pharo-Infinity/src';
    baseline: 'Infinity';
    load
```

### Examples

```Smalltalk
1 * Infinity negative = Infinity negative
```
